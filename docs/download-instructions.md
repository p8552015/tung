## 📱 處理流程

### 步驟 1: 下載原圖
1. 訪問 Unsplash/Pexels/Pixabay
2. 搜尋推薦關鍵字
3. 選擇高解析度版本下載
4. 確認為商用免費授權

### 步驟 2: 編輯處理
```
調整尺寸 → 統一色調 → 添加文字 → 品牌元素 → 格式轉換
```

#### 統一色調設定
- **主色**: #0066CC (科技藍)
- **飽和度**: +10-20%
- **對比度**: +10-15%
- **亮度**: 微調至適中

#### 文字元素
- **字體**: Source Sans Pro 或 Noto Sans TC
- **顏色**: 白色或深灰 #333333
- **大小**: Logo 24-36px，橫幅 48-72px

### 步驟 3: 輸出設定
- **格式**: WebP (最佳) 或 PNG (後備)
- **品質**: 85-90%
- **壓縮**: 啟用
